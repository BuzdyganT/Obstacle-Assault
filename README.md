Projekt zrealizowany w ramach kursu "Unreal Engine 5 C++ Game Development (Fully Updated in 5.6)". Uczę się w nim podstaw silnika.


## Wymagane zasoby (Dependencies)

Ze względu na ograniczenia rozmiaru na GitHubie, ciężkie modele 3D i tekstury nie są dołączone do repozytorium. Aby projekt wyglądał i działał kompletnie, musisz pobrać następujące darmowe paczki z platformy Fab:

* **Stylized Eastern Village** (umieść w folderze `Content/Asian_Village`)
* **Construction Site VOL.1** (umieść w folderze `Content/Construction_VOL1`)
* **Survival Character FREE** (umieść w folderze `Content/Survival_Character/`)
